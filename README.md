## 🛠 Tech Stack

- **Framework:** React 18 + TypeScript
- **Build tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Forms / Leads:** ManyChat integration (Instagram / Messenger)
- **Deployment:** Vercel

## 🚀 Getting Started

**Requirements:** Node.js 18+

```bash
# Clone the repository
git clone https://github.com/nico-coudouy/<repo-name>.git
cd <repo-name>

# Install dependencies
npm install

# Run the development server
npm run dev
```

The site will be available at `http://localhost:5173`.

### Build for production

```bash
npm run build
npm run preview
```

## 💡 About Bot Pilot

Bot Pilot is an automation service that helps small and medium businesses handle customer conversations on Instagram and Facebook Messenger using ManyChat combined with generative AI APIs (Gemini, Groq). The service focuses on:

- Answering frequent questions 24/7
- Qualifying leads before handing them to a human
- Collecting contact information and appointment requests
- Integrating with CRM and WhatsApp flows
